Overview

This is my personal Visual Studio Code configuration optimized for Python, JavaScript, and web development. It includes carefully selected extensions, editor settings, and keybindings that enhance productivity and maintain code quality.

What's Included
  • Complete settings.json configuration
  • Recommended extensions list
  • Custom keybindings
  • Theme and appearance settings
  • Language-specific configurations

Essential Extensions

These extensions significantly improve the development experience:

General Development

  • Python (ms-python.python) - Python language support with IntelliSense
  • Pylance (ms-python.vscode-pylance) - Fast, feature-rich Python language server
  • ESLint (dbaeumer.vscode-eslint) - JavaScript linting
  • Prettier (esbenp.prettier-vscode) - Code formatter

Productivity

  • GitHub Copilot (github.copilot) - AI pair programmer
  • GitLens (eamodio.gitlens) - Git supercharged
  • Error Lens (usernamehw.errorlens) - Inline error highlighting
  • Live Server (ritwickdey.liveserver) - Local development server

Settings Configuration

settings.json 
{
  // Editor
  "editor.fontSize": 14,
  "editor.fontFamily": "'Fira Code', 'Courier New', monospace",
  "editor.fontLigatures": true,
  "editor.lineHeight": 1.6,
  "editor.tabSize": 2,
  "editor.insertSpaces": true,
  "editor.wordWrap": "on",
  "editor.formatOnSave": true,
  "editor.minimap.enabled": false,
  "editor.suggestSelection": "first",
  "editor.bracketPairColorization.enabled": true,
  "editor.guides.bracketPairs": true,

  // Python specific
  "[python]": {
    "editor.tabSize": 4,
    "editor.defaultFormatter": "ms-python.black-formatter",
    "editor.formatOnSave": true,
    "editor.codeActionsOnSave": {
      "source.organizeImports": "explicit"
    }
  },
  "python.linting.enabled": true,
  "python.linting.pylintEnabled": true,
  "python.formatting.provider": "black",

  // JavaScript/TypeScript
  "[javascript]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  },
  "[typescript]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  },

  // Files
  "files.autoSave": "onFocusChange",
  "files.trimTrailingWhitespace": true,
  "files.insertFinalNewline": true,

  // Terminal
  "terminal.integrated.fontSize": 13,
  "terminal.integrated.fontFamily": "'Fira Code', monospace",

  // Workbench
  "workbench.colorTheme": "One Dark Pro",
  "workbench.iconTheme": "material-icon-theme",
  "workbench.startupEditor": "newUntitledFile",

  // Git
  "git.autofetch": true,
  "git.confirmSync": false,
  "git.enableSmartCommit": true
}

Installation Guide

  1. Open VS Code Settings
    Press Ctrl+, (Windows/Linux) or Cmd+, (Mac) to open settings.
  2. Access settings.json
    Click the "Open Settings (JSON)" icon in the top-right corner of the settings window.
  3. Copy configuration
    Replace or merge the contents with the configuration above.
  4. Install extensions
    Go to Extensions view (Ctrl+Shift+X) and search for each extension by name.
  5. Restart VS Code
    Reload the window for all changes to take effect.
