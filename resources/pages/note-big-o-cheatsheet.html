Big O Cheat Sheet

Type: Note
Category: Algorithms
Date Added: October 15, 2025
algorithms complexity big-o reference

Overview

Big O notation describes the performance or complexity of an algorithm. This cheat sheet provides quick reference tables for common data structures and algorithms, along with their time and space complexities.

Complexity Classes (from best to worst)
  • O(1) - Constant time
  • O(log n) - Logarithmic time
  • O(n) - Linear time
  • O(n log n) - Linearithmic time
  • O(n²) - Quadratic time
  • O(2ⁿ) - Exponential time
  • O(n!) - Factorial time

Rules of Thumb

  1. Drop constants: O(2n) becomes O(n)
  2. Drop non-dominant terms: O(n² + n) becomes O(n²)
  3. Different inputs use different variables: O(a + b) not O(n)
  4. Nested loops multiply: Two nested loops → O(n²)
  5. Consecutive operations add: O(a) + O(b) = O(a + b)
