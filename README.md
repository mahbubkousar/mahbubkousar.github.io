## Project Structure

```
.
├── index.html          # Main HTML file
├── style.css          # Main stylesheet
├── fonts/            # Font directory
│   ├── LM-bold-italic.{ttf,woff,woff2}
│   ├── LM-bold.{ttf,woff,woff2}
│   ├── LM-italic.{ttf,woff,woff2}
│   ├── LM-regular.{ttf,woff,woff2}
│   ├── Libertinus-bold-italic.woff2
│   ├── Libertinus-bold.woff2
│   ├── Libertinus-italic.woff2
│   ├── Libertinus-regular.woff2
│   ├── Libertinus-semibold-italic.woff2
│   └── Libertinus-semibold.woff2
├── prism/            # Prism.js for syntax highlighting
│   ├── prism.css    # Prism stylesheet
│   └── prism.js     # Prism JavaScript
├── LICENSE           # License file
└── README.md         # Project documentation
```

## Components

- **Main Files**: The root directory contains the main `index.html` and `style.css` files that define the structure and styling of the portfolio.

- **Fonts**: The `fonts` directory includes two font families:
  - Latin Modern (LM) fonts in various formats (TTF, WOFF, WOFF2)
  - Libertinus fonts in WOFF2 format

- **Syntax Highlighting**: The `prism` directory contains Prism.js files for code syntax highlighting functionality:
  - `prism.css` for styling code blocks
  - `prism.js` for syntax highlighting functionality
